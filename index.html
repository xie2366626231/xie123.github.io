有界性

上界：∃k1

假设f(x)这个定义域最大为300，那么就是k1大于等于这个值
f(x) <= k1

下界：∃k2

f(x) >= k2
最后f(x)结果永远小于k2

有界：∃k1>=3 ∃k2 <=3

k1 >= f(x) <= k2
那么f(x)=3，因为k1作为上界,他的值为3 到 无限，k2，为下界3 到 -无限,根据表达式得出他为3

无界：∃m

m < f(x1), m2 = ∀
m他作为一个下界，m2作为一个任意数

单调性

x1 < x2 f(x1) < f(x2)  单调增
x1 < x2 f(x1) > f(x2)  单调减

奇偶性D关于圆点对称

f(-x) = f(x) 偶
f (-x) = f(-x) 奇

周期性

∃正数i

f(x + i) = f(x)
最小周期 y = sin x的最小周期2π

并非每个周期函数都有最小周期

这个说实话贼抽像，电脑画图不太好画，所以就不上图了

不存在最小的有理数作为周期
假设：
D(x) = 1  x ∈ Q   有理数
D(x) = 0  x ∈ Qc  无理数
任何x + r的有理数相加都数一个有理数，所以不存在最小的周期
x + 有理数 + r = 有理数
x + 无理数 + r = 无理数

Jinja2模板语言介绍

jinja2是一个python广泛使用的模板引擎，是由python语言实现的模板语言，他的设计思想来源于Django的模板引擎，并扩展了其语法及一系列的强大功能，尤其是flask框架内置的模板语言

由于django默认模板引擎功能不齐全，速度慢，所以我们也可以在Django中使用jinja2，他的速度比Django默认的模板引擎快10-20倍

jinja2语法

在jinja2中，存在三种语法 

1.控制结构 {% %}
2.变量取值 { {} }
3.注释{ # # }
jinja2模板中使用{ {} }语法表示，他是一个变量，它是一种特殊的占位符，当利用janja2进行渲染的时候，它会把这些特殊的占位符进行填充/替换，jinja2支持python中所有的Python数据类型比如列表、字段、对象等。

过滤器

变量可以通过”过滤器”进行修改，过滤器可以理解为是jinja2里面的内置函数和字符串处理函数。

只需要在变量后面使用管道|分割，多个过滤器可以链式调用，前一个过滤器的输出会作为后一个过滤器的输入。

我反正一般不用这玩意，都是在代码里面进行修改的
