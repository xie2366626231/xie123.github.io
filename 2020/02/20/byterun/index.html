字节跳动面试题

发表于 2020-02-20 分类于 Javascript

今天朋友去面试字节跳动前端挂了，面试官给出了道题，emmm我帮他分析一梭子

开始解题

题目如下，这道题面试官问控制台分别输出的值是那三个，并且问你逻辑步骤，我刚刚拿到这道题的时候放眼看了一下 个人认为是0，1，3，最后朋友告诉我说是3，6，9然后我就思考了一会emmm

经过我的调试 i等于3，可能有人会疑问i=3 i<3代码应该不成立，的确他并没有执行代码并且for循环他执行了三次 i分别为 1，2，3依次进行赋值，通过debug调试，分别执行了result的三个对象引用，total在代码中进行累加，正常逻辑来说 33 33 3*3，但是最终控制台输出结果是3，6，9，看完下图其实就懂了，这个题我也懵逼了一huo。

总结

通过这坑其实我发现了一个有趣的事情，js的变量权重引用，在函数里2个同样的名字会优先采用局部变量的值,这个事情也告诉我们写代码需要规范。
