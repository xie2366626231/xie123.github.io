函数的几种特征
字数统计: 297 个字,阅读时间约: 1分钟。

有界性

上界：∃k1

假设f(x)这个定义域最大为300，那么就是k1大于等于这个值
f(x) <= k1

下界：∃k2

f(x) >= k2
最后f(x)结果永远小于k2

有界：∃k1>=3 ∃k2 <=3

k1 >= f(x) <= k2
那么f(x)=3，因为k1作为上界,他的值为3 到 无限，k2，为下界3 到 -无限,根据表达式得出他为3

无界：∃m

m < f(x1), m2 = ∀
m他作为一个下界，m2作为一个任意数

单调性

x1 < x2 f(x1) < f(x2)  单调增
x1 < x2 f(x1) > f(x2)  单调减

奇偶性D关于圆点对称

f(-x) = f(x) 偶
f (-x) = f(-x) 奇

周期性

∃正数i

f(x + i) = f(x)
最小周期 y = sin x的最小周期2π

并非每个周期函数都有最小周期

这个说实话贼抽像，电脑画图不太好画，所以就不上图了

不存在最小的有理数作为周期
假设：
D(x) = 1  x ∈ Q   有理数
D(x) = 0  x ∈ Qc  无理数
任何x + r的有理数相加都数一个有理数，所以不存在最小的周期
x + 有理数 + r = 有理数
x + 无理数 + r = 无理数
